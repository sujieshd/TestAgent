# TestAgent

This repository contains a Microsoft Copilot Studio (MCS) agent project exported as `.mcs.yml` assets, including topics, actions, entities, and knowledge.

## Structure

- `agent.mcs.yml` — agent definition
- `settings.mcs.yml` — agent settings
- `connectionreferences.mcs.yml` — connection references
- `topics/` — conversation topics
- `actions/` — action definitions
- `entities/` — entities used by the agent
- `knowledge/` — knowledge sources and files

## Getting started

Open this project in Microsoft Copilot Studio and import the agent assets as needed.

## Notes

- This repo stores configuration and content files only.
- The `.mcs` folder contains local metadata used by the tooling.
